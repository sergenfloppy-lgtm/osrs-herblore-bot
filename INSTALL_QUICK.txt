OSRS HERBLORE BOT - QUICK START
================================

WHY IT WON'T RUN ON THIS SERVER:
- No pip (package manager)
- No GUI/display
- No OSRS installed
- Can't control your mouse/keyboard remotely

TO RUN THIS BOT:
1. Download to YOUR gaming computer (Windows/Mac/Linux with OSRS installed)
2. Install Python 3.11+
3. Run: pip install -r requirements.txt
4. Run: python main.py
5. Follow setup wizard

GITHUB: https://github.com/sergenfloppy-lgtm/osrs-herblore-bot

This is a DESKTOP AUTOMATION tool - it needs to run where you play OSRS!
